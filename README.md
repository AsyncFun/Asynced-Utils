# Asynced-Utils
A project where I try to make an all in one utility discord bot in python.

# Important Note
When using a fork, replace the "SECRET_TOKEN" in the .env file to your bot's secret token, double quotation marks excluded.
*NEVER share your discord token to anyone. They can have full control of your bot if they have the token. If you have shared it, just reset the token from Discord's Developer's Portal.
